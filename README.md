# E-commerce Data Analysis Project (Excel)
-----------------------
## 📊 Project Overview
This project analyses an e-commerce dataset of 10,000+ orders to explore sales performance, revenue distribution, and customer behaviour.
The analysis was conducted using Microsoft Excel with pivot tables, charts, KPIs, and interactive slicers.

## 🔬 Dataset Information
- Source: Kaggle (public dataset, used for educational purposes)
- Records: 10,000+ rows
- Time period: 2024
- Currency: Converted from EGP to EUR for consistency

## 🎯 Objectives (Analysis)
- Identify key revenue-driving product categories
- Analyse sales performance by platform and location
- Examine monthly revenue trends
- Explore relationships between sales volume, ratings, and reviews

## 🛠️ Methodology
- Data cleaning and validation (missing values, data consistency checks)
- Feature creation (monthly breakdown, average order value)
- Pivot-based aggregations and KPI calculations
- Interactive dashboard design using slicers

## 💡 Key Insights
- Electronics and Fashion account for over 60% of total revenue.
- Revenue is evenly distributed across platforms, indicating low channel risk.
- Monthly revenue remains stable throughout the year with no strong seasonality.
- Higher review volume does not necessarily correlate with higher average ratings.
- Brand sales volumes are relatively balanced, with no dominant brand.

## 📜 Deliverables
- Interactive Excel dashboard with dynamic KPIs
- Revenue and performance analysis by category, platform, brand, and city
- Business-focused insights to support decision-making
